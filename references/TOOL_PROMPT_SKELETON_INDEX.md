# Tool Prompt Skeleton Index

Use these prompt skeletons after the brand has already been defined.

## Read order

1. `REFERENCE_STYLE_DISTILLATION.md`
2. `BRAND_FOUNDATION_DESIGN.md`
3. `UI_UX_DESIGN.md` when relevant
4. `APPLICATION_DESIGN.md`
5. `DESIGN_INDEX.md`
6. `TOOL_ADAPTER_INDEX.md`
7. matching adapter file
8. matching prompt skeleton
