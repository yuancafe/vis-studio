# Figma MCP Adapter

Use this adapter when building or refining brand-aligned work in Figma through MCP.

## Priorities

- layout and structure
- component relationships
- typography hierarchy
- spacing and grid logic
- token-consistent styling

## Read first

- `BRAND_FOUNDATION_DESIGN.md`
- `UI_UX_DESIGN.md` for digital tasks
- `APPLICATION_DESIGN.md` for collateral-style frames

## Emphasize

- sections, frames, and hierarchy
- auto-layout behavior
- component families and variants
- reusable style and token logic

## Avoid

- vague atmosphere-only prompts
- bitmap-heavy instructions without layout structure
- asking Figma MCP to invent the brand from scratch
