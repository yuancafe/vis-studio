# Pencil Adapter

Use this adapter for fast concepting, wireframe-to-design translation, or screen exploration where the tool behaves like a UI-focused design agent.

## Priorities

- UI layout
- screen flow
- content hierarchy
- tone of components

## Read first

- `BRAND_FOUNDATION_DESIGN.md`
- `UI_UX_DESIGN.md`

## Emphasize

- screen purpose
- key sections and modules
- hierarchy and rhythm
- interaction tone

## Avoid

- overloading with print-production constraints
- asking for identity invention instead of identity application
