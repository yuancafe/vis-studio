# Canva Adapter

Use this adapter for template-safe design work, repeated variants, and simple production across many sizes.

## Priorities

- template logic
- repeatable layouts
- editable text regions
- export-ready multi-size systems

## Read first

- `BRAND_FOUNDATION_DESIGN.md`
- `APPLICATION_DESIGN.md`

## Emphasize

- clear template zones
- text density limits
- size adaptation rules
- safe areas and repeatability

## Avoid

- overcomplex bespoke composition
- Illustrator-grade vector expectations
- relying on Canva for technical identity construction
