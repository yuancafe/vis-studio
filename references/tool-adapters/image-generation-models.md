# Image Generation Model Adapter

Use this adapter for tools and models such as Doubao, Kling, Nano Banana, or other text-to-image / image-edit generation systems.

## Priorities

- atmosphere
- composition
- color and material cues
- subject styling
- guardrails against brand drift

## Read first

- `BRAND_FOUNDATION_DESIGN.md`
- `APPLICATION_DESIGN.md`
- `REFERENCE_STYLE_DISTILLATION.md`

## Emphasize

- exact visual mood
- subject and scene setup
- brand color restraint
- type treatment expectations if text will be added later
- what must be avoided

## Avoid

- asking the model to reproduce a reference brand literally
- trusting generative text rendering for final typography
- using the model alone for technical brand system work

## Best use cases

- campaign key visuals
- poster backgrounds
- mood imagery
- stylized supporting graphics
- concept frames for later refinement in design tools
