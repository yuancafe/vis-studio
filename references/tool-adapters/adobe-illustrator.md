# Adobe Illustrator Adapter

Use this adapter for vector-heavy identity and application work.

## Priorities

- logo construction
- geometry and alignment
- vector pattern systems
- packaging outlines
- signage and print vector assets

## Read first

- `BRAND_FOUNDATION_DESIGN.md`
- `APPLICATION_DESIGN.md`

## Emphasize

- precise geometry
- lockup rules
- stroke and corner behavior
- scalable vector output
- manual-vector boundaries

## Avoid

- pretending Illustrator-heavy work is fully automated
- relying on atmosphere-only prompts for technical vector work
- mixing mockup language into vector construction tasks
