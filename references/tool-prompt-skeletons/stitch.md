# Stitch Prompt Skeleton

Use this skeleton when directing Stitch.

```text
Use the bundled brand files as the source of truth.

Task:
- Design [page/view/flow] for [brand/use case].

Read first:
- BRAND_FOUNDATION_DESIGN.md
- UI_UX_DESIGN.md
- DESIGN_INDEX.md
- TOOL_ADAPTER_INDEX.md
- tool-adapters/stitch.md

Output goal:
- Generate a digital interface that matches the approved brand and interaction tone.

Key constraints:
- Visual atmosphere: [insert]
- Layout behavior: [insert]
- Navigation pattern: [insert]
- CTA tone: [insert]
- Components to emphasize: [insert]
- Avoid: [insert]

Execution notes:
- Prioritize module structure, hierarchy, responsive behavior, and interface clarity.
- Do not fallback to generic SaaS styling.
```
