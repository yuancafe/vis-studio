# Figma MCP Prompt Skeleton

Use this skeleton when directing Figma MCP.

```text
Use the bundled brand files as the source of truth.

Task:
- Create or refine [screen/frame/component/system] for [brand/task].

Read first:
- REFERENCE_STYLE_DISTILLATION.md
- BRAND_FOUNDATION_DESIGN.md
- UI_UX_DESIGN.md
- DESIGN_INDEX.md
- TOOL_ADAPTER_INDEX.md
- tool-adapters/figma-mcp.md

Output goal:
- Build a Figma result that stays faithful to the approved brand system.

Key constraints:
- Brand posture: [insert]
- Typography behavior: [insert]
- Color logic: [insert]
- Grid and spacing: [insert]
- Component tone: [insert]
- Avoid: [insert]

Execution notes:
- Use clear frame hierarchy and auto-layout where relevant.
- Prefer reusable components and variants.
- Do not invent a new visual language.
```
