# Image Generation Models Prompt Skeleton

Use this skeleton for Doubao, Kling, Nano Banana, and similar image-generation or image-edit models.

```text
Use the bundled brand files as the source of truth.

Task:
- Generate [poster/key visual/background/concept image/supporting graphic] for [brand/use case].

Read first:
- REFERENCE_STYLE_DISTILLATION.md
- BRAND_FOUNDATION_DESIGN.md
- APPLICATION_DESIGN.md
- DESIGN_INDEX.md
- TOOL_ADAPTER_INDEX.md
- tool-adapters/image-generation-models.md

Output goal:
- Produce a brand-aligned visual that can be used directly or refined later in design tools.

Prompt guidance:
- Subject: [insert]
- Scene/composition: [insert]
- Lighting/materials: [insert]
- Palette and atmosphere: [insert]
- Typography expectation: [insert whether text should be omitted or added later]
- Avoid: [insert]

Execution notes:
- Do not copy a reference brand literally.
- Treat generated typography as non-final unless the model is only producing a background or concept image.
```
