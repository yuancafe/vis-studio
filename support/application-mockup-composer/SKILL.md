# Application Mockup Composer

## Overview

Handle the mockup-led route for brand applications that need contextual presentation rather than pure template reuse.

## Use This Skill

- stationery previews
- notebook and merch previews
- packaging presentation
- environmental scenes
- image-led contextual applications

## Workflow

### 1. Read the brand pack and recipe

Load the relevant rules before composing anything.

### 2. Confirm route

Only proceed for `mockup_semi_auto` items.

### 3. Use Photoshop-oriented production

Use the Photoshop route to:

- prepare imagery
- adapt color and composition
- build realistic mockups
- explain fit in context

### 4. Record boundaries

If the piece crosses into precise vector master-art, stop and hand off a manual brief.
