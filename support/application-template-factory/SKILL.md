# Application Template Factory

## Overview

Handle the scalable template route for brand applications.

## Use This Skill

- social templates
- banners
- website covers
- deck covers
- repeated collateral with stable layout logic

## Workflow

### 1. Read the brand pack

Load the approved tokens, logo rules, type rules, and application recipe.

### 2. Confirm template suitability

Only proceed for `template_auto` items.

### 3. Use Canva-oriented production

Use the Canva route to:

- create or choose templates
- adapt sizes
- build variants
- keep outputs within the approved system

### 4. Record what was produced

Write back:

- sizes covered
- variant count
- export formats
- any manual follow-up needed
