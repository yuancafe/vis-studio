# Application Families

Use these families when normalizing the scope matrix.

- `social`
- `banner`
- `web`
- `deck`
- `document`
- `stationery`
- `packaging`
- `merch`
- `signage`
- `wayfinding`
- `environment`

Choose the nearest useful family. Avoid inventing one-off families unless the work truly needs its own lane.
