# Brand Pack Schema

Required top-level keys:

- `brand`
- `brand_guideline`
- `brand_tokens`
- `logo_system`
- `style_playbook_selection`
- `application_recipes`
- `asset_manifest`

Recommended metadata:

- `version`
- `generated_at`
- `owners`
- `notes`

Use snake_case in the serialized JSON output.
