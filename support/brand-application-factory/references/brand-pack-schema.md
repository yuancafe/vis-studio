# Brand Pack Schema

Required top-level keys:

- `brand`
- `brand_guideline`
- `brand_tokens`
- `logo_system`
- `style_playbook_selection`
- `application_recipes`
- `asset_manifest`

Recommended metadata:

- `version`
- `generated_at`
- `owners`
- `notes`
- `reference_style_distillation`
- `design_md_documents`

Recommended `design_md_documents` keys:

- `brand_foundation_design`
- `ui_ux_design`
- `application_design`
- `design_index`

Use snake_case in the serialized JSON output.
