# Scope Bands

Use these bands to keep application work realistic and priced by actual volume.

- `Core` = 1-10 items
- `Growth` = 11-30 items
- `System` = 31-100 items
- `Enterprise` = 100+ items

## Rules

- `Core`: can be handled as one planned bundle
- `Growth`: requires family grouping and template reuse
- `System`: requires route classification and phased delivery
- `Enterprise`: requires batching, forced prioritization, and a backlog
