# Milestone Gates

Use hard approval gates. Do not allow endless drift.

## 25% gate

Approve:

- route diagnosis
- discovery findings
- integrated audit
- strategic problem statement

No visual exploration beyond rough references before this gate.

## 50% gate

Approve:

- concept directions or stylescapes
- overall creative direction

No detailed production system before this gate.

## 75% gate

Approve:

- high-fidelity core identity assets
- structural logo decisions
- color and typography decisions

## Final gate

Approve:

- guidelines
- assets
- rollout materials
- governance plan

## Feedback rules

- centralize feedback
- convert vague comments into explicit decisions
- use milestone framing, not endless rolling opinion
