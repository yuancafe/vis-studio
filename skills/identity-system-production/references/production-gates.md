# Production Gates

## Must pass before approval

- logo works in black and white
- logo works small
- responsive logo logic exists where needed
- type hierarchy is readable and controlled
- color values are fully specified
- applications have been tested on real contexts

## Red flags

- too many type families
- color palette chosen only for mood
- mark depends on gradients or fine detail to work
- system collapses when real content enters
- design decisions cannot be defended in business terms
