# Foundational Application Proof

Use a small proof set to validate the identity system before the application phase expands.

Recommended proof set:

- avatar or favicon
- website hero or banner
- social post cover
- slide cover
- one simple print touchpoint

The goal is not volume. The goal is to prove:

- logo reduction works
- color roles survive scale changes
- typography hierarchy still reads in context
- the graphic system can extend without breaking
