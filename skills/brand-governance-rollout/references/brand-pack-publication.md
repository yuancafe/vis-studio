# Brand Pack Publication

Publish the final `brand-pack` only after the identity and application layers are approved.

## Required sections

- `brand-guideline`
- `brand-tokens`
- `logo-system`
- `style-playbook-selection`
- `application-recipes`
- `asset-manifest`

## Release rules

- freeze approved values before export
- version the pack
- record owner and update date
- do not publish incomplete sections as placeholders

## Wrapper handoff

Use the published pack as the only source for generating the brand-specific application wrapper skill.
