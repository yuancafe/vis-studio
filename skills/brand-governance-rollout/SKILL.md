# Brand Governance Rollout

## Overview

Handle the delivery and operationalization phase. Convert the approved identity and application outputs into a usable operating system: Brand Identity Guideline, brand-pack publication, wrapper-skill handoff, asset organization, rollout logic, and ongoing governance.

## Why This Skill Exists

NotebookLM synthesis treats governance and rollout as a distinct phase. The creative work is not complete when the logo is approved; it is complete when the brand can be applied consistently by real teams and reused safely in future automation.

## Entry Requirement

Begin only after application approval from `brand-application-system`.

## Workflow

### 1. Build the living guideline

Document:

- brand summary
- logo rules
- color rules
- typography rules
- imagery rules
- application rules
- dos and don'ts

Treat this as the final Brand Identity Guideline, not a generic style sheet.

### 2. Package the asset system

Define:

- source files
- export files
- naming logic
- access location

### 3. Finalize the brand pack and wrapper handoff

Publish the finalized machine-readable pack:

- `brand-guideline`
- `brand-tokens`
- `logo-system`
- `style-playbook-selection`
- `application-recipes`
- `asset-manifest`

Use `brand-application-factory` to prepare the wrapper-skill handoff for future application generation.

### 4. Prepare rollout materials

Create or specify:

- templates
- priority applications
- launch or transition checklist
- phase-two backlog

### 5. Establish governance

Define:

- brand guardian / owner
- approval path
- vendor onboarding
- audit cadence

## Outputs

- Brand Identity Guideline
- finalized brand pack
- wrapper-skill handoff
- asset and export structure
- rollout kit
- governance checklist

## Non-Negotiables

- Never bury rules in a hard-to-find static file
- Never leave file ownership and access ambiguous
- Never launch externally before internal alignment is ready
- Never treat guideline creation as purely aesthetic packaging
- Never publish a wrapper skill against an unstable brand pack
