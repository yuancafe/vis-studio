# DESIGN_INDEX.md

## Purpose

This folder contains the AI-readable design documents for the brand.

## Read Order

1. `REFERENCE_STYLE_DISTILLATION.md`
2. `BRAND_FOUNDATION_DESIGN.md`
3. `UI_UX_DESIGN.md` when digital scope exists
4. `APPLICATION_DESIGN.md`

## Authority Map

- `REFERENCE_STYLE_DISTILLATION.md`: why the style direction exists
- `BRAND_FOUNDATION_DESIGN.md`: system foundation and stable brand rules
- `UI_UX_DESIGN.md`: digital interface patterns and interaction behavior
- `APPLICATION_DESIGN.md`: non-UI touchpoint and collateral rules

## Tool Usage

- Stitch: foundation first, then UI/UX
- Figma MCP: foundation first, then UI/UX or applications depending on task
- Pencil or similar: foundation plus application rules

## Update Rule

When the brand pack changes, these files must be reviewed and updated together.
